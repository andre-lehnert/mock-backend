# Flyway-Datenbankmigration

## Getting started

Führe 

```mvn clean flyway:clean flyway:migrate -PLOCAL``` 

im ``bogenliga-db-migration`` Modul aus, um das Datenbank-Schema anzulegen und mit Testdaten zu befüllen.
